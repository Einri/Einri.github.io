print()是Python提供的一个方法，主要用来输出。

a = 1 就是把1赋值给a， 程序中遇见 = ，是说右边计算的值赋给左边。

字符串

字符串只需要加上双引号，双引号是英文下输入 。

拼接字符串

字符串拼接只要在他们中间添上+就行 。

字符串变量

1是整数，所以a是整型变量。变量的类型取决于存储的值的类型。a = “Hello”

第三课

顺序结构

程序是按照从上到下顺序执行的，多个语句也是一样的。

条件结构if

条件结构是用来帮助我们作比较的。只有当括号里的条件成立时，才会执行if的代码块。在Python中， 通过空格缩进来确定该语句是否属于代码块中的一部分。

条件结构if else

if 括号里的条件满足时，执行 if 对应的代码块 ，输出“苹果数大于2个”。if 括号里的条件不满足时， 执行else对应的代码块，输出“苹果数小于2个”。

顺序结构和选择结构混合

第四课

列表的定义

scores就是一个列表，里面的 90，78，82就是score这个列表里面的元素(element)
在Python中，用方括号[]来表示列表，用逗号来分隔其中的元素。

如果元素是字符(String)的话，元素要加双引号。mom，dad，me，dog就是family这个列表里的元素。

下标访问列表元素

在列表中，每一个元素都有一个索引(index) ，在Python中，第一个列表元素的索引是0而不是1 。

修改列表中的元素

通过索引访问该值，并通过变量重新赋值。

添加元素到列表

使用append()方法给列表添加元素。

第五课

int()将字符串或数字转换

int()只能取整数而并不能四舍五入。

max()&min()求最大值和最小值

len()求列表的长度

列表的长度指的是它有多少个元素。

abs()求绝对值

什么是绝对值？正数（比0大的数）和0的绝对值是它本身，负数（比0小的数 ）的绝对值是它的反数 。”绝对的”英文是absolute，所以abs就是取absolute的前几个字母，成为一个简称。

pow()返回x的y次方的值

在python里，我们能用pow(x,y)表示x的y次方。

使用系统函数range()自动

range()函数规定一个范围。range(1,5)指的就是一个数字范围，包括元素1，2，3，4，不包括5。

range()可以用来制作列表。

累加列表

就是把列表里的元素一个一个相加。比如一个列表 a = [1,2,3] 累加的话就是1+2+3 = 6 。

利用range()来算一下1至10的累加。

求平均值

用平均数公式： 平均数 = 和／个数， 得到平均数。

累称列表

把一个数字列表里的所有元素相乘，在python里我们用星号* 表示乘，在累乘中 a 的初始值是1。

列表里统计出现次数

打印100以内的偶数

while条件…执行

例：利用while循环显示8-1，从8开始至1的八个数字。

布尔变量、逻辑真假值

布尔变量(Boolean Variable)有两种逻辑状态真(True) 和假(False)，真也可以用 1 表示，假的可以用 0 表示。 ==是一个比较判断符号，表示等于，在需要判断真假的情况下使用。
例：判断一下列表 list = [“a”,”b”,”c”,”d”] 的长度是否为3。

例：判断一下列表 list = range(11,20) 的长度是否为9。

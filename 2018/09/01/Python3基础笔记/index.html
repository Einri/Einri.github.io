字典

python里的字典（Dictionary）是一种数据类型，字典可以存储任何类型的对象。python中字典的意思和新华字典差不多，比如我们查一个不认识的字，字典里会有对这个字的解释。字典包含键(key)和值(value)，格式像下面这样：dict = {key1 : value1, key2 : value2 }
字典中的键必须是唯一的，值可以不唯一！值可以取任何数据类型，但键必须是不可变的。
像字符串，数字或元组这样不可变的才能是键。列表不能作为字典里的键。
字典里的元素没有顺序，所以不能通过下标引用元素。
字典里同一个键不能出现两次。同样的键，最后一个会覆盖先前的值。
